# 포챔스 정보 서비스 경쟁 분석 — A1 산출물 (2026-07-05)

## 총괄
- 웹 "세컨드 스크린"(팀빌더/데미지계산/티어) 시장은 **10개+ 서비스 포화 레드오션**: op.gg, Pikalytics, ChampTeams.gg, PikaChampions, Porygon Labs, pkmnchamps.com, championsbattledata.com, Champions Builder, Pokebase, ChampionsHub.gg, ChampDex, Champions Lab, TurnZeroVGC…
- **"게임 화면 위 실시간 오버레이" 폼팩터는 경쟁자 전무** (유사 모바일 오버레이 앱 미발견) → 우리의 유일 포지션.
- 결론: 웹 서비스 카테고리(팀빌더 등)로 후발 진입은 무의미. **보유 DB를 오버레이에서 즉시 소비 가능한 짧은 정보로 채우는 것**이 전략.

## 서비스별 요점
- **op.gg**: Pokedex/Tier(순위형, 싱글·더블 분리, 시즌 변동)/Speed(동속 그룹핑·성격/부스트/아이템 반영·내/상대 토글)/Sample Builds(309개, 사용률 표시 없음)/Replica Teams(521팀, 팀코드)/Team Builder.
- **championsbattledata.com**: 사용률 + **고급 검색 문법**(`Fake Out, spe>=100`, `move<=5=Earthquake`), API 제공.
- **Pikalytics**: 포챔스 완전 지원(Reg M-B, S3). 팀빌더(사용률 기반 추천), 정밀 데미지 계산기(웨더/필드/메가), Speed Tiers, Top Teams, 대회 집계(Limitless/RK9/VR). 가장 데이터 통합 폭넓음.
- **ChampTeams.gg**: 가장 기능 풍부한 경쟁자 — Battle Mode(상대 팀 붙여넣으면 픽 추천+실시간 스피드바), 트릭룸/윙달 반영 스피드, 주간 티어.
- **PikaChampions**: SP(노력치) 최적화 + 팀 약점 자동 경고(속도컨트롤 부재/인티미데이트 부재 감지).
- **Victory Road**: 포챔스 리플리카 팀 + 자체 대회(Champions Arena). LimitlessVGC는 아직 Gen9 위주.

## 오버레이 관점 분류
- **오버레이 결정적**: 타입/특성/기술 사용률(보유), **스피드 비교**(종족값 보유·로직 필요), 아이템 사용률(보유), 테라/메가 확률(보유), **파트너 사용률 기반 팀 예측**(보유·경쟁사 미제공 차별화), 4배약점 배지(보유).
- **앱 내 별도 화면용**: 팀빌더, 정밀 데미지계산, 티어 탐색, 리플리카 팀, 고급 검색.
- **불필요**: 대회 스탠딩 트래킹, Showdown import.

## Top 5 우선순위 (가치/난이도/DB보유)
1. **기술 사용률 Top 4~6** — 최고가치/난이도低/보유 (이미 카드에 있음 — 강화만)
2. **스피드 비교(최종 실속 비교, 내 파티 입력 필요)** — 매우 높음/중간/종족값·성격·아이템 보유
3. **아이템 사용률 + 메가 확률 표시** — 높음/低/보유
4. **파트너 사용률 "예상 팀원" 힌트** — 중상/중간/보유 · **유일 차별화 각도**
5. **4배 약점 경고 배지** — 중상/최저/보유 (확장패널에 있음 — 카드 단계로 승격 후보)

차순위 보류: 정밀 데미지 계산기(입력 UX 부적합·경쟁 극심), 팀빌더(레드오션), 샘플빌드 탐색.

## 추가 심층 조사 (2차)
### 한국 서비스 (중요)
- **포케챔스(yodams)**: 도감·계산기 5종·견제범위·파티상성·**선출 추천**. 안드로이드 앱 정식 출시.
- **포케샘플**: 통계/샘플/파티/빌더/교환/계산기/스피드/티어메이커, 계정제.
- **스마트누오**: 개인 개발 계산기 — **포챔스 출시 후 일간 사용량 ~30배 급증**(수요 검증 실증). 안드로이드 앱, 광고 없음 표방.
### 근접 선례
- **포챔스 헬퍼**(PC 전용, dcinside): 화면 인식으로 필드상태(랭크/날씨/순풍) 자동 반영 데미지 계산. 모니터 2대 필요 — **오버레이 아님, PC 전용** → 모바일 공백 유지.
- **DualScreenDex**(GitHub, MIT): Kotlin+ML Kit OCR로 구세대 포켓몬 게임 화면 인식(AccessibilityService, 듀얼스크린 핸드헬드용). 기술 스택 거의 동일한 오픈소스 — 구현 참고 가치.
- **ChampDex**(iOS): 도감/퀴즈 등 풍부하나 오버레이·OCR 없음.
- op.gg 데스크톱 오버레이는 포챔스 미지원(LoL/TFT 등 5종만).
### 인접 장르 교훈
- Porofessor/OP.GG 데스크톱은 전부 **게임사 공식 라이브 API** 의존 — 포챔스엔 그런 API가 없어 **OCR 화면인식이 유일 경로**(우리 접근 타당성 재확인). 단 화면에 없는 정보(상대 전적)는 원리적으로 불가.
### 결론 보강
- 웹 도구 20개+ 난립(도감+계산기+팀빌더 5종세트가 표준). 차별화 사례: PokeSynergy(설명가능 추천), PikaChampions(Meta Survival), Chouten(MCP 서버), championsbattledata(API).
- **모바일 실시간 오버레이 = 시장 공백 확정. 수요는 스마트누오 30배 급증으로 검증됨.**
