> 2026-07-05 · 근거: [research/](research/) 5종 (경쟁분석·Play요건·IP리스크·배포·크래시리포팅)

## 1. 포지셔닝 (조사 결론)
- 웹 "세컨드 스크린"(도감+계산기+팀빌더)은 **20개+ 서비스 포화** — 진입 금지.
- **모바일 실시간 오버레이는 시장 공백 확정**(최근접 선례도 PC 전용/타 게임). 수요는 스마트누오 30배 트래픽 급증으로 검증.
- 전략: **보유 DB(사용률·아이템·파트너·종족값)를 게임 중 즉시 소비 가능한 짧은 정보로 오버레이에 채우는 것**에 집중.

## 3. 운영급 UI/UX (P33)
- MainActivity를 **Material 3 운영 수준**으로: 다크테마, 구조화된 온보딩(단계 카드), 설정 그룹화, 브랜드 일관성.
- 로컬 `ui-ux-pro-max` 스킬(Compose 지원) 참조해 디자인 토큰 정립.
- 스킬 참고: 외부 오픈소스 스킬 설치는 이번 세션에서 겪은 프롬프트 인젝션 리스크 + 세션 내 스킬(ui-ux-pro-max, superpowers)로 충분하다는 판단으로 보류(정직 기록). A3 조사 2회가 레이트리밋으로 소실된 것도 사실.

## 4. 출시 준비물 (P34)
- **법적**: 개인정보처리방침(ko/en, "No data collected" — 온디바이스·전송0 실체 그대로) GitHub Pages 게시 + 앱 내 링크. 비공식 팬메이드 고지(닌텐도 등 무관) 온보딩+스토어 문구. 오픈소스 라이선스 화면(ML Kit 등).
- **네이밍**: "포챔스 서포터/PokeChamps Supporter" 유지(Poke Genie 패턴 — 완전명 "Pokémon" 미사용). 공식 아트워크 0 유지.
- **크래시**: UncaughtExceptionHandler → 로컬 저장 + 설정 "버그 리포트 공유"(ACTION_SEND). SDK 0, 전송 0 원칙 유지.
- **배포 (사이드로드 우선)**: GitHub Release v0.2.0(서명 APK) + README 설치 가이드 + **Obtainium 배지** + 앱 내 "새 버전 확인"(P13 manifest에 apk versionCode 확장).
