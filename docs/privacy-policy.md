## 1. 수집하는 데이터

**없음 (No data collected).**

포챔스 서포터는 다음 어느 것도 수집·저장·전송하지 않습니다.

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 위치 정보
- 연락처·사진·파일 접근
- 광고 식별자, 기기 식별자
- 사용 통계·분석(analytics)·행동 추적
- 계정 정보 (**앱에 계정·로그인 개념 자체가 없습니다**)

---

## 2. 화면 캡처(MediaProjection)의 사용 범위

이 앱은 게임 위에 정보를 표시하기 위해 안드로이드의 화면 캡처(MediaProjection) 권한을 사용합니다. 이 권한은 사용자가 매번 명시적으로 동의해야만 활성화됩니다.

- 캡처된 화면 프레임은 **기기 메모리 안에서만** 처리되어, 화면에 표시된 **상대 포켓몬 이름 텍스트를 온디바이스 OCR(광학 문자 인식)로 읽는 데에만** 사용됩니다.
- **캡처된 이미지는 저장되지 않으며(0), 어디로도 전송되지 않습니다(0).** 다음 프레임이 오면 이전 프레임은 즉시 폐기됩니다.
- OCR 처리는 앱에 내장된 Google ML Kit 온디바이스 모델로 이뤄지며, **인터넷에 접속하지 않습니다.**
- 이 앱은 게임 메모리를 읽거나, 게임 서버와 통신하거나, 입력을 자동화하지 않습니다. **화면에 이미 공개된 정보만** 사용합니다.

---

## 3. 인터넷 사용 범위

이 앱은 딱 한 가지 목적으로만 인터넷을 사용할 수 있습니다.

- **포켓몬 데이터 업데이트(선택·수동)**: 사용자가 설정에서 "데이터 업데이트" 버튼을 직접 누를 때에 한해, GitHub Pages(정적 파일 호스팅)에서 포켓몬 타입/특성/사용률 등의 **정적 JSON 파일을 내려받습니다.** 이 과정에서 앱은 서버로 어떤 개인정보도 보내지 않으며, 단순한 파일 다운로드만 발생합니다.
  - 이때 GitHub의 서버가 표준 웹 요청 로그(IP 주소 등)를 남길 수 있으나, 이는 **GitHub의 인프라 소관**이며 앱 개발자는 그 로그에 접근하지 않고 수집하지도 않습니다. GitHub의 개인정보 처리에 대해서는 GitHub 개인정보처리방침을 참고하세요.
- 이 버튼을 누르지 않으면 앱은 인터넷에 접속하지 않으며, **앱에 내장된 데이터만으로 완전히 오프라인 동작**합니다.

---

## 4. 크래시(오류) 로그

앱이 예기치 않게 종료(크래시)될 경우, 원인 파악을 돕기 위해 오류 정보(스택 트레이스, 앱 버전, 기기 모델명)를 **기기 내부 저장소에만 로컬로 저장**합니다.

- 이 로그는 **자동으로 전송되지 않습니다.** 앱 개발자를 포함해 누구에게도 자동으로 전달되지 않습니다.
- 사용자가 설정 > 고급 > "버그 리포트 공유"를 직접 눌러 공유(이메일·메시지 등)를 선택할 때에만, 사용자가 고른 앱으로 로그 텍스트가 전달됩니다. **공유 여부와 대상은 전적으로 사용자가 결정합니다.**
- 로컬에 저장되는 크래시 로그는 최근 5개까지만 유지되며, 개인 식별 정보를 담지 않습니다(기기 모델명은 예: "Galaxy S23" 수준의 일반 정보).

## 7. 권한 목록과 사유

| 권한 | 사유 |
|---|---|
| 다른 앱 위에 표시(SYSTEM_ALERT_WINDOW) | 게임 위에 정보 카드 오버레이 표시 |
| 화면 캡처(MediaProjection, 매번 동의) | 상대 이름 텍스트를 온디바이스 OCR 로 읽기 (저장·전송 없음) |
| 포그라운드 서비스 | 캡처 중 상태바 알림 유지 (안드로이드 규정) |
| 알림(POST_NOTIFICATIONS) | 캡처 중임을 알리는 상태바 알림 |
| 인터넷(INTERNET) | 사용자가 "데이터 업데이트"를 누를 때 정적 JSON 다운로드만 |
